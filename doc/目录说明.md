# HMI 项目代码目录详细说明

本文档对 `hmi` 项目下的每一个源代码文件及资源文件进行了详细说明，旨在帮助开发者快速理解项目结构与各模块职责。

## 1. 根目录
- [main.py](file:///home/pi/project/hmi/main.py): 项目入口程序。负责初始化全局日志、加载配置、创建 `QApplication` 并启动主界面。
- [config.json](file:///home/pi/project/hmi/config.json): 核心配置文件。定义了 MQTT 服务器地址、API 接口基础路径及其他运行参数。
- [requirements.txt](file:///home/pi/project/hmi/requirements.txt): Python 依赖列表。列出了项目运行所需的第三方库（如 PyQt6, paho-mqtt 等）。
- [init.sql](file:///home/pi/project/hmi/init.sql): 数据库初始化汇总脚本。
- [err.log](file:///home/pi/project/hmi/err.log): 系统运行时产生的错误日志记录文件。
- [mqtt-connection-guide.md](file:///home/pi/project/hmi/mqtt-connection-guide.md): MQTT 连接调试指南。
- [API文档.md](file:///home/pi/project/hmi/API%E6%96%87%E6%A1%A3.md): 项目接口定义说明。

---

## 2. 基础设施层 (infrastructure/)
封装了与外部系统（网络、硬件、系统工具）交互的底层逻辑。
- [__init__.py](file:///home/pi/project/hmi/infrastructure/__init__.py): 包初始化。
- [http_client.py](file:///home/pi/project/hmi/infrastructure/http_client.py): 封装基于 `requests` 的 HTTP 请求工具类，处理 Token 认证及异常。
- [mqtt_client.py](file:///home/pi/project/hmi/infrastructure/mqtt_client.py): 基于 `paho-mqtt` 实现的异步客户端，负责主题订阅与消息发布。
- [thread_pool.py](file:///home/pi/project/hmi/infrastructure/thread_pool.py): 全局线程池封装，用于处理 UI 线程外的耗时任务。

---

## 3. 领域模型层 (domain/)
定义核心业务实体，不依赖于任何框架逻辑。
- [__init__.py](file:///home/pi/project/hmi/domain/__init__.py): 包初始化。
- [device.py](file:///home/pi/project/hmi/domain/device.py): 设备实体类（如 ID、状态、类型等）。
- [material.py](file:///home/pi/project/hmi/domain/material.py): 物料信息实体。
- [warehouse.py](file:///home/pi/project/hmi/domain/warehouse.py): 仓储库位实体。
- [workflow.py](file:///home/pi/project/hmi/domain/workflow.py): 生产工作流/任务实体。

---

## 4. 服务层 (services/)
实现具体的业务逻辑流转，调用基础设施层并返回给视图模型。
- [__init__.py](file:///home/pi/project/hmi/services/__init__.py): 包初始化。
- [auth_service.py](file:///home/pi/project/hmi/services/auth_service.py): 登录认证、权限校验相关逻辑。
- [device_service.py](file:///home/pi/project/hmi/services/device_service.py): 设备的增删改查及状态同步逻辑。

---

## 5. 视图模型层 (viewmodels/)
作为 View 与 Service 的中间件，处理 UI 展示逻辑和数据状态。
- [__init__.py](file:///home/pi/project/hmi/viewmodels/__init__.py): 包初始化。
- [base_viewmodel.py](file:///home/pi/project/hmi/viewmodels/base_viewmodel.py): 所有 ViewModel 的基类，提供通用的信号处理机制。
- [device_viewmodel.py](file:///home/pi/project/hmi/viewmodels/device_viewmodel.py): 负责设备列表、详情的数据转换与 UI 指令分发。
- [login_viewmodel.py](file:///home/pi/project/hmi/viewmodels/login_viewmodel.py): 负责登录页面的输入验证与状态转换。

---

## 6. 视图层 (views/)
基于 PyQt6 实现的所有图形界面及组件。
- [__init__.py](file:///home/pi/project/hmi/views/__init__.py): 包初始化。
- [main_window.py](file:///home/pi/project/hmi/views/main_window.py): 应用程序主框架，包含导航栏和内容堆栈。
- [login_dialog.py](file:///home/pi/project/hmi/views/login_dialog.py): 用户登录对话框。
- [production_overview.py](file:///home/pi/project/hmi/views/production_overview.py): 数字化车间生产看板页面。
- [smart_production.py](file:///home/pi/project/hmi/views/smart_production.py): 智能生产/运维主页面。
- [smart_production_canvas.py](file:///home/pi/project/hmi/views/smart_production_canvas.py): 核心组件：工作流可视化编辑器画布。
- [smart_production_dialogs.py](file:///home/pi/project/hmi/views/smart_production_dialogs.py): 生产页面相关的配置对话框（节点选择、参数配置等）。
- [smart_production_constants.py](file:///home/pi/project/hmi/views/smart_production_constants.py): 视图层专用的常量定义。
- [smart_production_utils.py](file:///home/pi/project/hmi/views/smart_production_utils.py): 视图层专用的 UI 辅助工具函数。

### 6.1 视图组件 (views/components/)
- [charts.py](file:///home/pi/project/hmi/views/components/charts.py): 各类数据可视化图表组件。
- [device_control_panel.py](file:///home/pi/project/hmi/views/components/device_control_panel.py): 设备控制快捷面板。
- [dialogs.py](file:///home/pi/project/hmi/views/components/dialogs.py): 通用弹出框组件。
