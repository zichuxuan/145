# PLC 模拟器 Modbus TCP 快速参考

## 1. 通信参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **Unit ID** | 1 | 从站地址（Device ID / Slave ID） |
| **IP 地址** | 127.0.0.1 | 根据实际配置修改 |
| **端口** | 5020 | Modbus TCP 默认端口 |
| **协议** | Modbus TCP | 基于 TCP/IP |

---

## 2. 功能码对照表

| 功能码 | 名称 | 用途 | 操作 |
|--------|------|------|------|
| **01** | 读线圈 | 读取控制线圈状态 | 读 |
| **02** | 读离散输入 | 读取设备状态反馈 | 读 |
| **04** | 读输入寄存器 | 读取状态码/位置值 | 读 |
| **05** | 写单个线圈 | 发送启停控制指令 | 写 |

---

## 3. 设备控制参数

### 3.1 电机控制

| 参数 | 值 | 说明 |
|------|-----|------|
| **Unit ID** | 1 | 从站地址 |
| **功能码** | 05 | 写单个线圈 |
| **启动地址** | 1 (偏移 0) | 第1个电机控制线圈 |
| **启动值** | 0xFF00 | True / ON |
| **停止地址** | 1 (偏移 0) | 同上 |
| **停止值** | 0x0000 | False / OFF |

**状态读取：**
| 功能码 | 地址 | 说明 |
|--------|------|------|
| 02 | 10001 (偏移 0) | 运行中状态 |
| 02 | 10002 (偏移 1) | 已停止状态 |
| 02 | 10003 (偏移 2) | 故障状态 |
| 04 | 30001 (偏移 0) | 状态码 (0=停止, 1=运行, 2=故障) |

### 3.2 阀门控制

| 参数 | 值 | 说明 |
|------|-----|------|
| **Unit ID** | 1 | 从站地址 |
| **功能码** | 05 | 写单个线圈 |
| **启动地址** | 101 (偏移 100) | 第1个阀门控制线圈 |
| **启动值** | 0xFF00 | True / 打开 |
| **停止地址** | 101 (偏移 100) | 同上 |
| **停止值** | 0x0000 | False / 关闭 |

**状态读取：**
| 功能码 | 地址 | 说明 |
|--------|------|------|
| 02 | 10101 (偏移 100) | 已打开状态 |
| 02 | 10102 (偏移 101) | 已关闭状态 |
| 02 | 10103 (偏移 102) | 故障状态 |
| 04 | 30101 (偏移 100) | 位置值 (0-100%) |

## 5. 通信报文示例

### 5.1 启动电机1（功能码05）

**请求报文：**
```
[事务标识] 00 00
[协议标识] 00 00
[长度]     00 06
[Unit ID]  01
[功能码]   05
[地址高]   00
[地址低]   00        (地址1的偏移0)
[值高]     FF        (启动值高字节)
[值低]     00        (启动值低字节)
```

**完整十六进制：** `00 00 00 00 00 06 01 05 00 00 FF 00`

---

## 6. 代码示例

### Python (pymodbus)

```python
from pymodbus.client import ModbusTcpClient

# 参数配置
UNIT_ID = 1
MOTOR1_COIL = 1      # 电机1控制地址

client = ModbusTcpClient("127.0.0.1", port=5020)
client.connect()

# 启动电机（功能码05，值0xFF00）
client.write_coil(MOTOR1_COIL - 1, True, slave=UNIT_ID)

# 停止电机（功能码05，值0x0000）
client.write_coil(MOTOR1_COIL - 1, False, slave=UNIT_ID)

# 读取状态（功能码02）
result = client.read_discrete_inputs(0, count=3, slave=UNIT_ID)
print(f"运行:{result.bits[0]}, 停止:{result.bits[1]}, 故障:{result.bits[2]}")

# 读取状态码（功能码04）
result = client.read_input_registers(0, count=1, slave=UNIT_ID)
print(f"状态码:{result.registers[0]}")  # 0=停止, 1=运行, 2=故障

client.close()
```
